# close-other-tabs

Chrome Extension to add Close Other Tabs functionality.

## Note

This extension was written when _Close Other Tabs_ functionality disappeared from Chrome for me.  It now seems that I was part of an A/B test that didn't get rolled out, and _Close Other Tabs_ is here to stay.

As a result, this extension doesn't have much benefit other than as a code example.

## About

~~To replace the item recently removed from the right-click menu on Chrome tabs,~~ this extension does three things:

* It adds a button to the toolbar you can click on to close other tabs.
* It adds a Close Other Tabs item to the standard right-click context menu.
* It adds the `Alt+Shift+W` keyboard shortcut.

The extension adds a toolbar button and adds an item to the right click context menu you get when clicking on the page rather than ~~restoring the menu item that was removed~~ adding a new menu item because Extensions cannot add items to the tab menu.
